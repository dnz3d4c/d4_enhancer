# /tts

TTS 로그 분석 후 패턴 문서 업데이트.

## 실행 절차

1. **로그 파일 확인**
   - 위치: `D4GameEnhancer.log` (프로젝트 루트)
   - 없으면 사용자에게 알림

2. **패턴 분석**
   - 총 라인수 카운트
   - 주요 패턴별 빈도 측정:
     - 퍼센트만 (`^\[RAW\] '[0-9]+%'$`)
     - 에테르 관련 (`에테르`)
     - 빈 문자열 (`^\[RAW\] ''$`)
     - 버튼 UI (`버튼`)
     - 보호막 (`보호막:`)
     - 몬스터명 (`추방된`)
     - 기타 새로운 패턴

3. **문서 업데이트**
   - 파일: `docs/SPEECH_PATTERNS.md`
   - 분석 날짜, 총 라인수 갱신
   - 새 패턴 발견 시 테이블에 추가
   - 비율 재계산

4. **결과 보고**
   - 주요 변화 요약
   - 새로 발견된 패턴 강조

## 출력 예시

```
분석 완료: 10,388줄

상위 패턴:
- 진행률 숫자: 1,724 (16.6%)
- 에테르 메시지: 1,714 (16.5%)

새 패턴 발견: 없음

docs/SPEECH_PATTERNS.md 업데이트됨
```
