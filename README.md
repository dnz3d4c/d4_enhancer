# D4GameEnhancer

디아블로 IV용 NVDA 추가 기능. 게임 내 텍스트를 스크린 리더로 읽어준다.

## 설치

1. [Releases](https://github.com/dnz3d4c/d4_enhancer/releases)에서 `.nvda-addon` 다운로드
2. 파일 실행해서 NVDA에 설치

## 사용법

- 디아블로 IV 실행하면 자동으로 게임 텍스트 읽어줌
- `NVDA+ALT+D`: 디버그 로깅 켜기/끄기

## 빌드

```bash
uv run python scripts/build.py
```
